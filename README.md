# Game Trading API

## Setup & Installation
1. Install Dependencies:
```
pip install -r requirements.txt
```

2. Database setup:
* Create the database structure: Run app.py to initialize the database.

```
python app.py
```

* Run populate.py to add sample users, games and trades
```
python populate.py
```

## Database info
* Database: SQLite version: 
* ORM: Flask-SQLAlchemy
